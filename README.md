# Distributed Order Management System

A Java-based microservice-style application that demonstrates order and inventory
management using REST APIs, layered architecture, and Dockerized services.

## Features
- Order and inventory services separated by responsibility
- REST-based inter-service communication
- Thread-safe in-memory data handling
- Dockerized services for consistent deployment
- Unit testing for service validation

## Tech Stack
- Java
- Spring Boot
- REST APIs
- Docker
- Git
